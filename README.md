# üìÅ File-Based Employee Management System (Python)

This project is a **menu-driven Python application** that performs basic CRUD (Create, Read, Update, Delete) operations on a **text file** to manage employee records.  
It is built using **core Python concepts only**, with a strong focus on **file handling** and **string formatting**.

This project is intended for **learning and practice**, not for production use.

---

## üöÄ Features

- Create a structured text file with column headers
- Insert employee records into the file
- Read and display all stored records
- Update a specific field using Employee ID
- Replace any word throughout the file
- Delete a specific employee record
- Delete the entire file
- Simple command-line menu interface

---

## üß± Data Format

Each employee record contains the following fields:

- ID
- First Name
- Last Name
- Age
- Salary
- Email

Data is stored in a formatted table inside a `.txt` file.
